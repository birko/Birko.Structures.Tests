# Birko.Structures.Tests

Unit tests for Birko.Structures data structures.

## Test Coverage

- **Trees/AVLTreeTests** - AVL tree in-order traversal and balancing
- **Trees/BinarySearchTreeTests** - BST in-order traversal and operations

## Test Framework

- xUnit 2.9.3
- FluentAssertions 7.0.0
- .NET 10.0

## Running Tests

```bash
dotnet test Birko.Structures.Tests
```
